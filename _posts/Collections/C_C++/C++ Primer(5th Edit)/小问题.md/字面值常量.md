1. `3.14L` 表示什么类型？
    - `3.14` 默认表示 `double` 类型
    - `3.14f` 表示 `float` 类型
    - `3.14L` 表示 `long double` 类型
2. `1E-3F` 表示什么？
    - 等价于 `(1E-3)F` 
    - 表示 `(1*10^-3)F`
3. `10.` 表示什么？
    - C++ 环境下凡是带小数的字面量自动认为是double型的，另外，这一类常量小数点前后孤单的0可以省略，基本上几个常用的主流编程语言都是如此，比如0.2就可以写作.2，同样的10. 代表双精度浮点数10.0
    - 单精度浮点数(float)
    - 双精度浮点数(double)

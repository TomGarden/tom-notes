## 一、右侧的线
Settings -> Editor -> Preferred Line Length
- Identifies the length of a line which is used when wrapping text with the `soft Wrap At Preferred Line Length` setting enabled,in number of characters.

上述值设置一个较大的数字，线就看不见啦

## 二、自动换行
View -> Taggle Soft Wrap

## 三、拼写检查
对当前页面关闭拼写检查
- Menu -> Packages -> Spell Check -> toggle

关闭拼写检查功能对所有页面都不检查
- Menu -> Settings -> Packages -> spell-check -> Disable

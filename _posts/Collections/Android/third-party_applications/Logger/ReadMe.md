- 来者是是因为想要编写全局异常处理库,全局异常处理库需要将日志输出到本地.
- 个人认为 Logger 库可以更优雅
- "更优雅" 就是说调用的时候可插拔的功能类似 RxJava 的函数式调用
- 所以我们要研究下 RxJava 的原理和函数式调用的实现细节和实现思想.
